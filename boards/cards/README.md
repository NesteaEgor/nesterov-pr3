# Карточки задач

Карточка является задачей и содержит следующие элементы: название, описание, сроки, приоритет, метки, чек-листы, исполнители.


## Что можно делать в карточке
- редактировать поля
- добавлять чек-лист
- прикреплять файлы и изображения
- писать комментарии
